# My Personal Website V2

## Requirement
- Node v24.15
- pnpm v11.1

## Instalation
1. install depedencies `pnpm install`
1. run in dev mode `pnpm run dev`

## Deployment
1. install depedencies `pnpm install`
1. build the project `pnpm run build`
1. check is build working `pnpm run preview`
1. follow tanstack start deployment strategy [tanstack start deployment docs](https://tanstack.com/start/latest/docs/framework/react/guide/hosting) based on where do you want to deploy the application
